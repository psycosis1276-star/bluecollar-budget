# Blue Collar Budget

A **stupid simple** paycheck-based budget tool made for blue collar workers.

No fancy spreadsheets. No complicated menus. Just straightforward numbers so you can see what you actually take home and where it's going.

### Features
- Calculate regular + overtime pay
- Automatic federal & state tax estimates
- 401k percentage deduction
- Track monthly expenses
- Big Purchase Calculator (tells you how many OT hours you need for that new rifle, tool, etc.)
- Monthly Overview (Income vs Expenses)
- Saves your data between uses
- Start New Month option to reset

### How to Use
1. Run `python main.py`
2. Follow the menu options
3. Keep it simple — that's the whole point

### Why I Built This
Most budget apps feel like they were made by accountants for accountants. This one was made by a millwright for regular working guys who are tired as hell at the end of the day.
